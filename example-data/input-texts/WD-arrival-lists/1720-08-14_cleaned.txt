Ankunft deren hoh- und niederen Stands-Personen. Den 10. Augusti / 1720. Schotten-Thor. Herr Graf St. Julian / komt von Walsee / log. im Hoisischen Hauß. Karntner-Thor. Herr von Dütigen / Haubtmann vom Althannischen Regiment / komt aus Hungarn / log. im weisen Lambel. Den 11. Dito. Herr von Ränfdelshoffen / Probst zu Strasburg / in Kärnten / komt von dannen / log. im wilden Mann. Herr Baron Poztazky / und Herr Haubtmann Alberti / vom Lothringis. Regiment / kommen aus Hungarn / log. in 3. Haasen. Den 12. Dito. Stuben-Thor. Herr Doctor van der Beck / komt aus der Kaiserlichen Wallachey / log. bey dem Radel. Den 13. Dito. Karntner-Thor. Herr Wolf Ehrenreich / Graf von Auersperg / komt von St. Pölten / log. im Matschacker Hof. Herr Vogel / Ritmaister vom Croyschen Regiment / komt aus Hungarn / log. bey der Heil. Dreyfaltigkeit.
