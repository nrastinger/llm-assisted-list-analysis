Ankunft derer Hoh- und Niedern Stands-Personen. Den 24. Septemb. 1712. Roten-Thurn. Herr Graf Scherschoffski / von Prag / komtlog. beyn 3. Hacken. Kärntner-Thor. Herr Sarasini, Cavalier, komt von Trient / log. im goldnen Hirschen. Den 25. Dito. Herr Wüttich / Currier / komt von der Reichs-Armee / geht ins Post-Haus. Stuben-Thor. Herr Graf Carl Zitschi / komt von Presburg / log. im Erdödischen Hauß. Den 27. Dito. Schotten-Thor. Herr Hof-Kammer-Rath Mora / komt von Salzburg / log. in der Kärntnerstrassen / im Zinngiesserischen Hauß.
