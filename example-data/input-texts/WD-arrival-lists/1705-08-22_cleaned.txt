Ankunfft aller Hoch- und niedrigen Stands-Personen. Den 19. Augusti 1705. Cärnter-Thor. Herr Obrist-Lieutenant Becker / kombt von Groß-Wardein / logirt im guldnen Adler. Herr Aloisius Beda Toldo, kombt auß der Schütt / logirt im rothen Apfel. Herr Graf Dietrichstein und Herr Graf Breuner / kommen von Brün / logiren im Dietrichsteinischen Hauß. Herr Obrist Lieutenant Maleniz, kombt von der Armee auß Jtalien / logirt im Matschacker Hof. Rotten-Thurn. Herr Hillebrand Hof-Cammer-Rath / kombt von München / logirt im Oelerischen Hauß. Den 20. dito. Cärntner-Thor. Herr Graf Docon, kombt von Grätz / logirt im wilden Mann. Stuben-Thor. Ein Kayserl. Currier / Hr. Derpeniz, kombt von Jhro Durchl. Hrn. Marggrafen von Baaden / log. im Post-Ambt. Rothen-Thurn. Herr Graf Fux / Würzburgischer Gesandter / kombt von dannen / logirt beyn untern Jesuitern.
